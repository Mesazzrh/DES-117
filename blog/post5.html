Web Design Research

Understanding Flexibility and Units

November 21, 2024

So I took a look at this site, REM? PX? Why not both? because I do want to know more this area of CSS. This article discusses the author's decision to switch from using only relative units (rem) to a mix of relative (rem) and absolute (px) units in their website's CSS. The author explores the implications of this change and compares the visual differences between the two approaches through the screenshots he/she created.

And after viewing this video, Why Is CSS So Weird?, I noticed that both of them emphasize the importance of responsive design. With the video discussing about CSS's adaptability, while the article is providing practical advices on the uses of relative units like "rem" to make the designs could work on different devices and screen sizes.

Throughout this course, I’ve gained a solid understanding of HTML and CSS, starting with the basics of structuring an HTML document using semantic tags like header, nav, and footer. Later on, I’ve also learned to enhance the visual presentation of these elements using CSS properties such as color, font, margins, and padding.

Making a responsive design has also been a key area in this course. By using media queries and scalable units like %, vw, vh, and so on. Using these, I could now make a design that adapts to various screen sizes, making them mobile-friendly.

Overall, I’ve progressed from creating simple static pages to designing more complex layouts with responsive and visually engaging elements. However, there's a lot more out there for me to discover and learn.
