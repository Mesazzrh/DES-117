Typography and Structure

This project focused on exploring layout possibilities for a provided set of text about the Alberini Speaker Series, an event featuring distinguished speakers, their talk titles, and dates. The challenge was to transform raw information into visually engaging and accessible designs while adhering to a consistent format.

Using design principles such as hierarchy, alignment, paragraph rules, use of negative/positive space, and contrast, I created multiple layout options to present the event details effectively. Each iteration aimed to balance aesthetic appeal with functionality, ensuring that the text was both visually striking and easy to read. The final layout was chosen based on its ability to deliver the message clearly while maintaining a professional and dynamic appearance.

Tools & Techniques:

  • Software: Adobe InDesign
  • Design Elements: Font choices, hierarchy, lines for division
  • Focus: Typography, alignment, visual hierarchy, and text readability
